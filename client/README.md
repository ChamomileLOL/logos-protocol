## 🚀 Deployment Instructions (For the Rational Mind)

### Material Prerequisites
* Node.js (v24 or later)
* Docker Desktop
* MongoDB Atlas Account

### Local Actuality

1. **Backend Ignition:**
    ```bash
    cd server
    npm install
    node server.js
    ```
2. **Frontend Ignition:**
    ```bash
    cd client
    npm install
    npm run dev
    ```

### Containerized Actuality
1. **Build the Cage:**
    ```bash
    docker build -t logos-protocol-app .
    ```
2. **Run the Cage:**
    ```bash
    docker run -p 5000:5000 -e MONGO_URI="your_mongodb_uri" logos-protocol-app
    ```
